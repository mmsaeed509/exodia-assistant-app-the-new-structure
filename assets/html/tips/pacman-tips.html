 Sync The pacman DataBase sudo pacman -Syyy or p-sync .
 install a package file from your local system sudo pacman -U or ipkg .
 Online Packages Search sudo pacman -Ss or search .
 Local Packages Search (Installed Packages) sudo pacman -Qs or search-local .
 Get Package Info sudo pacman -Qi or pkg-info .
 clear Packages add Pacman Cache sudo pacman -Scc or clr-cache .
 Remove Lock File sudo rm /var/lib/pacman/db.lck or unlock .
 Remove Package with its Dependencies sudo pacman -Rns or autoremove .
 Force Remove Package sudo pacman -Rdd .
